Проект "Примитивная база данных" - это инициализируемая программа, имитирующая работу реальной БД в рамках второго домашнего задания по дисциплине "Программирование на Python" студента группы М25-555 Шакурова Егора.

Инструкция по сборке проекта и запуску базы данных:

1) Установка зависимостей: make install
2) Установка пакетов: make package-install
3) Запуск программы (базы данных): make project
4) Сборка проекта: make build
5) Публикация сборки: make publish
6) Проверка кода в соответствии с ruff: make lint

Команды для взаимодействия с базой данных:
1) create_table <имя_таблицы> <столбец1:тип> <столбец2:тип> ... - Cоздать таблицу
2) drop_table <имя_таблицы> - Удалить таблицу
3) list_tables - Вывести список всех таблиц
4) insert into <имя_таблицы> values (<значение1>, <значение2>, ...) - Вставить запись в таблицу
5) select from <имя_таблицы> where <столбец> = <значение> - Выбрать запись таблицы
6) select from <имя_таблицы> - Выбрать все записи таблицы
7) update <имя_таблицы> set <столбец1> = <новое_значение1> where <столбец_условия> = <значение_условия> - Обновить (заменить) запись таблицы
8) delete from <имя_таблицы> where <столбец> = <значение> - Удалить запись из таблицы
9) info <имя_таблицы> - Вывести информацию о таблице
10) exit - Выйти из программы
11) help - Справочная информация (список команд)
